# อาหารเหนือป้าย่า — ระบบจัดการรายรับรายจ่าย 🍽️

เว็บแอปสำหรับบันทึก **รายรับ–รายจ่าย** และดู **กำไร** ของร้านอาหาร
ใช้งานได้ทั้ง **โทรศัพท์ / iPad / คอมพิวเตอร์** เก็บข้อมูลในเครื่อง (ไม่ต้องสมัครบัญชี) และใช้ได้แม้ไม่มีเน็ต

## ✨ ฟีเจอร์
- บันทึกรายรับ / รายจ่าย พร้อมหมวดหมู่ วันที่ และรายละเอียด
- **ช่องทางการเงิน:** รายรับแยก เงินสด / เงินโอน / รับโอนอื่นๆ · รายจ่ายแยก เงินสด / เงินโอน
- **ภาพรวม (Dashboard):** กำไรสุทธิ, รายรับ, รายจ่าย + สรุปแยกตามช่องทาง แยกช่วงเวลา (วันนี้ / 7 วัน / เดือนนี้ / ทั้งหมด)
- **กราฟ:** แท่งรายรับ–รายจ่ายรายวัน และวงกลมสัดส่วนรายจ่ายตามหมวด
- **รายการ:** ดูย้อนหลัง จัดกลุ่มตามวัน ค้นหา และกรองตามประเภท/เดือน
- **แนบรูปใบเสร็จ:** ถ่าย/แนบรูปใบเสร็จในแต่ละรายการ (บีบอัดอัตโนมัติ)
- **รายงาน:** สรุปกำไรรายเดือน 6 เดือนล่าสุด พร้อมยอดสด/โอน
- หมวดหมู่ปรับแต่งเองได้ (เพิ่ม/ลบ)
- **สำรอง/กู้คืนข้อมูล** เป็นไฟล์ และ **ส่งออก Excel/CSV**
- ดีไซน์แบบ responsive (sidebar บนจอใหญ่ · แถบล่างบนมือถือ) + รองรับโหมดมืดอัตโนมัติ
- ติดตั้งลงหน้าจอมือถือได้ (PWA)

## ▶️ วิธีเปิดใช้งาน

### ง่ายที่สุด
ดับเบิลคลิกไฟล์ `index.html` เพื่อเปิดในเบราว์เซอร์ — ใช้ได้ทันที

> หมายเหตุ: การเปิดด้วยไฟล์ตรงๆ (`file://`) ใช้งานได้ครบ ยกเว้นการ “ติดตั้งเป็นแอป (PWA)” ที่ต้องเปิดผ่านเซิร์ฟเวอร์ (ด้านล่าง)

### เปิดผ่านเซิร์ฟเวอร์ (แนะนำ — เพื่อใช้ PWA และทดสอบบนมือถือ)
ในโฟลเดอร์นี้ รันคำสั่ง:

```bash
npx serve .
```

แล้วเปิดลิงก์ที่ขึ้น (เช่น `http://localhost:3000`) ในเบราว์เซอร์
หากต้องการเปิดบนมือถือในวง Wi‑Fi เดียวกัน ให้ใช้ IP ของเครื่องแทน `localhost`

## 📱 ติดตั้งลงมือถือ (ให้เป็นเหมือนแอป)
- **iPhone (Safari):** ปุ่มแชร์ → “เพิ่มไปยังหน้าจอโฮม”
- **Android (Chrome):** เมนู ⋮ → “ติดตั้งแอป” หรือกดปุ่ม “ติดตั้งลงเครื่อง” ในหน้า “ตั้งค่า” ของแอป

## 💾 ข้อมูลของคุณ
- เก็บอยู่ใน **เบราว์เซอร์ของเครื่องนี้เท่านั้น** (localStorage)
- การล้างข้อมูลเบราว์เซอร์อาจทำให้ข้อมูลหาย — **แนะนำให้กด “สำรองข้อมูล” เป็นประจำ**
- ย้ายเครื่อง/เปลี่ยนมือถือ: ใช้ “สำรองข้อมูล” แล้ว “นำเข้าข้อมูล” ในเครื่องใหม่

## 📂 ไฟล์ในโปรเจกต์
| ไฟล์ | หน้าที่ |
|------|---------|
| `index.html` | หน้าหลัก |
| `styles.css` | สไตล์ (รองรับโหมดมืดอัตโนมัติ) |
| `app.js` | ตรรกะทั้งหมด: state, การคำนวณ, กราฟ, ฟอร์ม |
| `manifest.webmanifest`, `sw.js`, `icon.svg` | ไฟล์สำหรับ PWA / ออฟไลน์ |

## 🔜 ต่อยอดได้ในอนาคต (อยู่ระหว่างตัดสินใจ)
- **เข้าระบบด้วย LINE + ซิงค์ข้อมูลขึ้น Cloud** (ใช้ได้หลายเครื่อง/หลายสาขา) — ต้องมีโฮสติ้ง HTTPS + LINE Login + ฐานข้อมูลหลังบ้าน
- **อ่านใบเสร็จอัตโนมัติด้วย AI** — ถ่ายรูปแล้วให้ระบบอ่านยอด/วันที่ แล้วกรอกให้เอง (ส่วนแนบรูปทำเสร็จแล้ว เหลือเชื่อมต่อ AI อ่านข้อมูล)
- ส่งสรุปยอดรับ-จ่ายเข้า LINE อัตโนมัติ
- ตั้งงบประมาณรายเดือน + แจ้งเตือน
